### Topics covered are as follows 
1.  **Python Basics**
    - [x] : Background Part 1: I/O Devices, CPU, and Memory
    - [x] : Background Part 2: Computer Organization & Operating Systems
    - [x] : Programming Concepts: Why do we need programming languages?
    - [x] : Python Introduction: Overview, standards, and implementations
    - [x] : Execution: How Python programs are executed
    - [x] : Terminology: Common Python programming terms
    - [x] : Setup: Python installation and writing your first program
    - [ ] : Documentation: Using comments in Python


2.  **Variables and Data Types**
    - [ ] : Variable Management: Variables in Python and swapping two variables
    - [ ] : Introspection: id() and type() functions
    - [ ] : Data Structures Introduction:
    - [ ] : Lists Introduction
    - [ ] : Tuples in Python
    - [ ] : Sets in Python
    - [ ] : Dictionaries in Python
    - [ ] : Type Conversion: Implicit and explicit type casting


3. **Input and Output**
    - [ ] : Output: Deep dive into the print() function
    - [ ] : Input: Handling user data with the input() function

4. **Operators**
   - [ ] : Types of Operators:
   - [ ] : Arithmetic Operators
   - [ ] : Logical Operators
   - [ ] : Identity and Comparison Operators
   - [ ] : Membership Test Operators
   - [ ] : Bitwise Operators (Parts 1 & 2)
   - [ ] : Mathematical Applications: Finding the nth term of Arithmetic and Geometric Progressions, Sum of natural numbers, and finding the last digit of a number.

5. **Flow Control**
   - [ ] : Conditional Statements: if, else, and elif
   - [ ] : Practical Logic: Even-Odd checks, finding the largest of three numbers, and Leap Year logic
   - [ ] : Project: Building a basic calculator program

6. **Loops**  
   - [ ] : Iteration Types: while loops and for loops
   - [ ] : Tools: Using the range() function
   - [ ] : Control Statements: break and continue
   - [ ] : Nested Logic: Nested loops and pattern printing (Square, Triangle, Inverted Triangle, and Pyramid)
   - [ ] : Algorithmic Problems: Counting digits, Factorials, GCD, LCM, Fibonacci sequence, and Prime number checks (including optimizations)

7. **Functions**
   - [ ] : Fundamentals: Defining functions and how they work internally
   - [ ] : Arguments: Default, Keyword, and Variable-length arguments
   - [ ] : Data Handling: Parameter passing and returning multiple values
   - [ ] : Scope: Global variables in Python
   - [ ] : Logic: Prime factorization and finding the first digit of a number

8. **Strings**
   - [ ] : Basics: Introduction to strings and string operations
   - [ ] : Formatting: Escape sequences, raw strings, and formatted strings (f-strings)
   - [ ] : Manipulation: Slicing and built-in string methods

9. **Lists & Advanced Data Structures**
   - [ ] : List Operations: Slicing, comprehension, and common methods
   - [ ] : Sets and Dictionaries: Advanced usage and unique properties
   - [ ] : Tuples: Immutability and use cases

10. **Object-Oriented Programming (OOP)**
    - [ ] : Core Concepts: Classes and Objects
    - [ ] : Implementation: Creating and manipulating Python classes effectively
